# email2markdown.site

Site web pour décrire l'application email2markdown - un outil pour convertir les emails en format Markdown.

## Description

Ce site est construit avec Nuxt 3 et présente l'application email2markdown, ses fonctionnalités et ses cas d'utilisation.

## Installation

Installer les dépendances :

```bash
npm install
```

## Développement

Lancer le serveur de développement sur `http://localhost:3000` :

```bash
npm run dev
```

## Production

Construire l'application pour la production :

```bash
npm run build
```

Prévisualiser localement la build de production :

```bash
npm run preview
```

Ou générer un site statique :

```bash
npm run generate
```

## Structure du projet

- `/pages` - Pages de l'application (index, features, about)
- `/layouts` - Layout par défaut avec navigation
- `/components` - Composants Vue réutilisables
- `/public` - Fichiers statiques

## Technologies

- **Nuxt 3** - Framework Vue.js pour applications web
- **Vue 3** - Framework JavaScript progressif
- **TypeScript** - Superset typé de JavaScript
